- Design System: for consistent colors, typography, spacing, and layout

- Maintain consistent design using Design System guidelines

- Produce React/JSX/MDX code ready to drop into the Docusaurus project



Ensure all code is modular, maintainable, responsive, and consistent with the Axionics brand.
